# ASX-Stock-Analysis

Analysis of 5 ASX-listed stocks (CBA, BHP, CSL, MQG, WBC) from 2023 to 2026.

## What this project covers
- Historical price trends
- Daily returns
- Volatility comparison across stocks
- 30-day and 90-day moving averages

## Tools used
- Python
- yfinance
- pandas
- matplotlib
